# Nebula

Nebula is a Real-Time Code editor which allows multiple users to edit specific codebase at the same time.

> For additional query join [the Discord channel](https://discord.gg/N4YmxtXrYs)

## Features

- Inbuilt Voice Call
- User Authentication
- Shared Terminal

## Tech Stack

- **Frontend:** React, SCSS
- **Backend:** FastAPI
- **Database:** MySQL
